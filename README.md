# CatSay
Silly little PowerShell module that lets you make a cat say something with a sign. Like this:
```ps1
Catsay -Text "Hi!"
```
Output:
```
  ∧,,,∧
 (• ⩊ •)
╭──U U────╮
│   Hi!   │
╰─────────╯
```
(Inspired by [Cowsay](https://github.com/piuccio/cowsay)), except it's a lot worse. LOL.
